<h2 id="projeto">3. Descrição do Projeto</h2>

<p>
O projeto consiste no desenvolvimento de uma plataforma web de cashback
integrada com múltiplas lojas online.
</p>

<p>
Os usuários poderão realizar compras através da plataforma
e receber parte do valor gasto de volta.
</p>

<hr>

<h2 id="objetivo">4. Objetivo do Sistema</h2>

<ul>
<li>Facilitar compras online</li>
<li>Gerar economia para os usuários</li>
<li>Integrar diversas lojas em uma única plataforma</li>
<li>Criar um sistema funcional de cashback real</li>
</ul>

<hr>

<h2 id="problema">5. Problema Resolvido</h2>

<p>
Muitas pessoas compram online sem receber benefícios financeiros.
O sistema busca centralizar lojas parceiras e oferecer retorno financeiro
aos clientes por meio do cashback.
</p>
